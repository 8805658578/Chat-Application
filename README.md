🚀 Real-Time Chat Application

📌 Description

This is a real-time chat application where users can create chat rooms and communicate instantly. The system uses WebSocket technology to enable fast and live messaging between multiple users.

---

✨ Features

- Create and join chat rooms
- Real-time messaging using WebSockets
- Multiple users can chat simultaneously
- Fast and responsive user interface
- REST API integration for backend communication

---

🛠️ Technologies Used

Backend:

- Java (Spring Boot)
- WebSocket
- REST API

Frontend:

- React (Vite)
- HTML, CSS, JavaScript

Database:

- MySQL

---

📂 Project Structure

- Backend → Spring Boot Application
- Frontend → React Application

---

⚙️ Setup Instructions

🔹 Backend Setup

1. Clone the repository
2. Open project in IntelliJ IDEA
3. Configure MySQL database
4. Run Spring Boot application

🔹 Frontend Setup

1. Go to frontend folder
2. Install dependencies:
   npm install
3. Run the project:
   npm run dev
